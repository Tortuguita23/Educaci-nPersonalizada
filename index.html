Este espacio está reservado para adjuntar las actividades complementarias. Nombrad los archivos como se indica para que el envío llegue correctamente.

Los documentos se reciben de forma privada y no se comparte el acceso al alumnado.

Entrega de presentaciones

Presenta tu exposición utilizando el formato presentacion_TuNombre.

Ejemplo: presentacion_AnaLopez.pdf

El archivo se enviará directamente desde la web. Comprueba que el nombre comienza con presentacion_ seguido de tu nombre y apellidos.

Entrega de atributos personales

Actividad: analizar los atributos radicales de tu persona en este momento y generar una propuesta de mejora para todos ellos, o los que se considere. Guarda el documento como atributos_TuNombre.

Ejemplo: atributos_CarlaPerez.pdf

El archivo se enviará directamente desde la web. Comprueba que el nombre comienza con atributos_ seguido de tu nombre y apellidos.

Estudio de caso: María

Si deseas compartir tu análisis y propuesta, utiliza el formato Maria_TuNombre. No se ofrecerá feedback individual, aunque las ideas podrán inspirar la resolución colectiva.

Ejemplo: Maria_LuisGomez.pdf

El archivo se enviará directamente desde la web. Comprueba que el nombre comienza con Maria_ seguido de tu nombre y apellidos.
